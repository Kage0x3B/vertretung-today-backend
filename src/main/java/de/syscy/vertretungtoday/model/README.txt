In diesem Ordner befinden sich sehr viele "Modell"/Datenbank Entitäten Klassen, die einmal zum Daten speichern im Programm natürlich genutzt werden,
aber auch von JPA, einer Bibliothek die sich um das Verwalten der Datenbank kümmert, benutzt werden, um automatisch Tabellen zu erstellen
und so die Objekte direkt in die Datenbank zu serialisieren.
